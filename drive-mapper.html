Paso 1 — Configuración
Requiere un OAuth Client ID de Google Cloud.
Es un proceso de ~5 minutos que se hace una sola vez. Expande las instrucciones abajo.
¿Cómo obtener el Client ID?
  1. Ve a console.cloud.google.com y crea o selecciona un proyecto
  2. En APIs y servicios → Biblioteca, habilita Google Drive API y Google Sheets API
  3. Ve a Credenciales → Crear credenciales → ID de cliente OAuth 2.0
  4. Tipo de aplicación: Aplicación web
  5. En "Orígenes de JavaScript autorizados" agrega null (para archivo local)
  6. Copia el Client ID (termina en .apps.googleusercontent.com)
  7. En Pantalla de consentimiento, agrega tu correo como usuario de prueba
Se guarda en el navegador (localStorage)
